# 💰➕ | Geld hinzufügen

Befehl: **`/economy add`**

Optionen:
- **`user:`**  
  Der Nutzer welchem Geld hinzugefügt werden soll.
- **`amount:`**  
  Der Betrag welcher hinzugefügt werden soll.

<u>Beschreibung:</u>  
 Einem Nutzer einen bestimmten Betrag hinzufügen.

<u>Berechtigungen</u>  
Nur Administratoren (Owner) können diesen Befehl nutzen.
