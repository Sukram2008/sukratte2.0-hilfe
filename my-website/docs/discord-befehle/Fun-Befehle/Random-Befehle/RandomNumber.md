# 🔢❓ | Zahl: Sagt dir eine zufällige Zahl

Befehl: **`/random number`**

Optionen:
- **`min:`** [Optinonal]  
  Die kleinst möglichste Zahl. (Standard: 1)
- **`max:`** [Optional]  
  Die höchst möglichste Zahl. (Standard: 42)

<u>Beschreibung:</u>  
 Man kann sich eine Zufällig Zahl sagen lassen.

<u>Berechtigungen</u>  
 Alle Nutzer.
